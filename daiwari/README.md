# 台割作成ツール 使い方

依存ライブラリなし・ブラウザ内で完結する台割作成ツールです。`daiwari_tool.html` をブラウザで開くだけで使えます（サーバー不要）。

## できること

- 書名・1台あたりのページ数を入力し、項目（大扉・はじめに・各章など）を並べると、自動で台割（折・通し番号・ノンブル）を計算
- 各項目は「ページ数入力」または「開始・終了ページ指定」のどちらかで入力可能
- ノンブルのリセット（章ごとに1ページ目からにする等）
- 奇数ページ始まりの自動調整（白ページの自動挿入）
- CSVコピー／ダウンロード
- 罫線つきPDF（台割表）のダウンロード

## 1. 書誌情報

- **書名**：任意。CSV・PDFのファイル名やヘッダーに使われます。
- **1台あたりのページ数**：折の単位（既定16ページ）。
- **奇数ページ始まりを自動調整**：チェックすると、項目ごとの「奇数始まり必須」設定と組み合わさって、開始ページが偶数になる場合に自動で白ページを1ページ挿入します（詳細は4章）。

## 2. 項目リストの入力

表に1行ずつ項目を追加していきます。

| 列 | 内容 |
|---|---|
| 項目名 | 章タイトルなど |
| ページ数 / 開始–終了 | 3章で説明する2つの入力モード |
| ノンブルリセット | チェックするとこの項目の先頭でノンブルが1に戻る（前が「i, ii…」のローマ数字なら、以降はアラビア数字になる） |
| 奇数始まり必須 | チェックすると、この項目の開始ページが奇数になるよう調整対象になる（4章参照） |
| × | 行の削除 |

「＋項目を追加」で空の行を追加、「台割を計算」を押すと下の台割表に反映されます。

### 行の並び替え

行の順番は次のどちらかで変更できます。

- 行左端の「⠿」をドラッグして、動かしたい位置にドロップする
- 行右端の「▲」「▼」ボタンで1行ずつ上下に動かす（先頭・末尾では押せません）

ドラッグ&ドロップはマウス操作向けです。タッチ操作の環境ではうまく反応しないことがあるので、その場合は▲▼ボタンを使ってください。

### まとめて貼り付け

「まとめて貼り付けで入力する」を開くと、テキストエリアに1行1項目でまとめて入力できます。台割用テキストファイル（.txt/.csv/.tsv）をドラッグ&ドロップするか、「ファイルを選択」で読み込むこともできます。読み込んだ内容はテキストエリアにも反映されるので、その場で見直し・編集できます。

```
#書名: サンプル技術書
#1台: 16
大扉	1
権利	1
はじめに	3
!第1章	34-67
```

- タブまたはカンマ区切りで「項目名, ページ数」
- 項目名の先頭に `!` を付けるとノンブルリセット
- 項目名の末尾に `*` を付けると奇数始まり必須（例: `!第1章*　34-67`）
- ページ数のところに `34-67` のように書くと、開始・終了ページ指定モードとして読み込まれる（3章参照）
- 先頭が `#` の行は書誌情報として扱われます。`#書名: ○○○` `#1台: 16` と書いておくと、書名・1台あたりのページ数を自動で読み取ります（この記法が無いファイルもこれまで通りそのまま読み込めます）
- 「この内容を反映」を押すと、**既存の行はすべて置き換わります**
- 「現在の項目を台割テキストとしてコピー」「台割テキストをダウンロード」を押すと、今の項目リストと書誌情報を上記と同じ`#書名:`/`#1台:`形式のテキストとして書き出せます。書き出したテキストは、そのままこのツールで読み込み直せます（読み込む→スクリプトなどで加工する→また読み込む、というラウンドトリップができます）
- 行そのものが消えることはありません。ページ数が読み取れない・終了ページが開始ページより前になっている等の場合は、その行もそのまま読み込んだうえで、反映後にメッセージでどの項目が要確認かを知らせます（該当項目は項目テーブル上でも⚠が表示されます）

## 3. ページ数入力 vs 開始・終了ページ指定

各行の「範囲」／「P数」ボタンで、その項目だけ入力モードを切り替えられます。

- **ページ数入力モード（既定）**：ページ数だけを入力。ノンブルは前の項目からの続きで自動的に振られます。章数や大まかな見積もりの段階に向いています。
- **開始・終了ページ指定モード**：実際の原稿ですでにページが確定している場合、開始ページと終了ページをそのまま入力します。ページ数（＝終了−開始＋1）は自動計算され、ノンブルは「前の項目からの続き」より指定した開始ページが優先されます。

切り替えは項目ごとに独立していて、同じ行でページ数と開始・終了ページを同時に持つことはありません（どちらか一方）。

### ⚠（警告アイコン）について

開始・終了ページ指定モードの項目で、次のいずれかが起きると項目テーブルにその行の⚠が表示されます（値は自動修正されません。ユーザーに気づいてもらうためだけの表示です）。

1. 前の項目までの計算上のノンブルと、指定した開始ページが食い違っている（何ページ足りない／重複しているかを含めてツールチップに表示）
2. 「奇数始まり必須」がチェックされているのに、指定した開始ページが偶数（4章の白ページ自動挿入が働かないための注意）

## 4. 奇数ページ始まりの自動調整と白ページ

「奇数ページ始まり」を有効にすると、**ページ数入力モードの項目**にのみ、以下が自動で行われます。

- その項目の「奇数始まり必須」がチェックされている
- かつ、その項目の開始ページが偶数になる

という条件がそろったときだけ、直前に白ページを1ページ自動挿入して開始ページを奇数にします。

**開始・終了ページ指定モードの項目には、この自動挿入は行われません。** すでに開始ページが確定しているため、勝手に白ページを挟むと指定したページ番号とズレてしまうからです。同じ条件（奇数始まり必須がONで開始ページが偶数）に当てはまる場合は、自動挿入の代わりに⚠で知らせます。

### 開始・終了ページ指定モードで白ページを入れたいとき

その項目の入力欄の下にある **「＋白p」ボタン**を押すと、その項目の直前に「（白）・ページ数1・ページ数入力モード」の行を1行差し込みます。以降は普通の項目として、必要なら名前や位置を編集・削除できます。

※ 自動挿入された白ページは台割表でイタリック表示＋「－奇数始まり調整」の注記が付きますが、この手動追加の白ページ行には付きません（見た目だけの違いで、台割としての機能に差はありません）。

## 5. 台割の見方・出力

- **折**：物理的なページ順で16ページ＝1折として繰り上がる番号（ノンブルのリセットとは無関係）
- **通し番号**：先頭からの物理ページ順
- **ノンブル**：ノンブルリセットを反映した実際の印字ページ番号（リセット前は小文字ローマ数字）
- **CSVをコピー／ダウンロード**：折・通し番号・ノンブル・内容をCSVで出力
- **PDFをダウンロード（罫線つき）**：ブラウザの印刷機能を使わず、pdf-libで直接組み立てた台割表PDF（日本語フォント埋め込み済み）。画面上の一覧はレビュー用の簡易表示です。

### 総ページ数が4の倍数かどうかのチェック

「台割を計算」を押すと、総ページ数（自動挿入・手動追加どちらの白ページも含めた実際のページ数）が4の倍数かどうかを自動でチェックします。4の倍数でない場合は、台割表の上に赤みがかったメッセージで「あと何ページ増やす／減らすと4の倍数になるか」を表示します。1台あたりのページ数（既定16）とは別の、印刷の基本単位に関するチェックです。

## 6. 想定する使い方の流れ

1. 企画段階：ページ数入力モードで各章のおおよそのページ数を入れて総ページ数・台数を把握
2. 原稿が固まってきたら、確定した章だけ開始・終了ページ指定モードに切り替えて実際のページ番号を入力
3. ⚠が出た行は、意図通りかどうかを確認（意図通りなら無視してよい）
4. 最終的にCSVかPDFで書き出して共有
